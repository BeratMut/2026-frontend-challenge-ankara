## Project Description

Bu proje, **Jotform API** üzerinden çeşitli form verilerini çekerek, bunları profesyonel bir dashboard arayüzünde görüntüleyen React tabanlı bir web uygulamasıdır.

**Ana Özellikleri:**

- 📋 **5 Farklı Form Verisi**: Checkins, Messages, Sightings, Personal Notes, Anonymous Tips
- 🔍 **Gelişmiş Arama ve Filtreleme**: Kişi adına göre araştırma yapabilme ve kategoriye göre filtreleme
- ⏱️ **Zaman Çizelgesi**: Seçilen kişinin tüm olaylarını kronolojik sırada görüntüleme
- 💳 **Profesyonel Kart Tasarımı**: Form türüne göre renklendirilmiş ve düzenli görünüm
- ⚡ **Hızlı Performans**: Vite build tool'u ve Tailwind CSS ile optimize edilmiş arayüz

**Teknoloji Yığını**: React 18+, Vite, Tailwind CSS, Jotform REST API

## Getting Started

### Gereksinimler

- Node.js 16+ ve npm
- Jotform API Key

### Kurulum Adımları

**1. Bağımlılıkları Yükleyin**

```bash
npm install
```

**2. `.env` Dosyası Oluşturun**

Proje kök dizininde `.env` dosyası oluşturun ve Jotform API Key'inizi ekleyin:

```env
VITE_API_KEY=sizin_api_key_numaraniz_buraya_gelecek
```

> **API Key Nasıl Alınır?**
>
> - Jotform hesabınıza giriş yapın
> - Ayarlar → API bölümüne gidin
> - API Key'inizi kopyalayın

**3. Geliştirme Sunucusunu Başlatın**

```bash
npm run dev
```

Tarayıcınızı açın ve `http://localhost:5173` adresine giderek uygulamayı kullanmaya başlayın.

**4. Üretim İçin Build Edin (İsteğe Bağlı)**

```bash
npm run build
```

### Kullanım

- 📋 **Checkins, Messages, Sightings, etc.** sekmelerine tıklayarak form verilerini görüntüleyin
- 🔍 **Arama ve Filtreleme Yap** sekmesinde kişi araması yapın
- ⏱️ **Zaman Akışı** sekmesinde kronolojik timeline görmek için kişi adı yazın
